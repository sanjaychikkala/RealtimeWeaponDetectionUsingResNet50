About the Project

This system implements real-time weapon detection and classification using advanced deep learning architectures (including ResNet-50 for image-level classification and YOLO-based annotation pipelines). It identifies dangerous objects such as Handgun, Automatic Rifle, Shotgun, SMG, Bazooka, Knife, Sword from live video streams and uploaded images/videos, providing instant bounding boxes and threat alerts to enhance security and public safety.
